WHAT HAPPENS DAILY
================================================================================

Time: 00:00 WIB (midnight)
Frequency: Every day

Process:
1. Task Scheduler wakes computer (if sleeping)
2. Runs run_daily_scraper.bat
3. Activates Python virtual environment
4. Runs complete pipeline:
   - Scrapes Instagram accounts
   - Extracts data with Gemini AI
   - Inserts into database
   - Cleans up expired opportunities
   - Verifies results
5. Deactivates virtual environment
6. Computer returns to previous state

Duration: 2-5 minutes typically
